## Welcome

Welcome to the Orgo API documentation. This API is built with Symfony and API Platform, providing a comprehensive REST API for managing adhesions, applications, users, and more.

The API follows REST conventions and supports JSON-LD, JSON, CSV, and multipart/form-data formats for most endpoints.

## Base URL

All API endpoints are prefixed with `/api/v1/`

## Content Types

The API supports multiple content types:
- `application/ld+json` (JSON-LD with Hydra)
- `application/json` (Standard JSON)
- `text/csv` (CSV format)
- `multipart/form-data` (For file uploads)

## Pagination

Collection endpoints support pagination with the following parameters:
- `page`: The collection page number (default: 1)
- `pagination`: Enable or disable pagination (boolean)
